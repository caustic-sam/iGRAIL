# Future Features

This file is the current forward-looking product list for iGRAIL.

## Near-Term Engineering Priorities

- scheduled publishing that actually executes on time, not just stores `scheduled_for`
- fuller author/profile joins in admin and public content responses
- quieter and more intentional observability instead of ad hoc console output
- dependency hygiene, including updating stale browser-baseline metadata

## Product Features Worth Building Next

- richer editorial workflow states and revision history
- stronger search and taxonomy browsing
- better analytics for readership and feed performance
- improved policy-source attribution and freshness tracking

## Experimental Or Longer-Horizon Items

- Web3-related publication experiments
- more advanced research tooling
- expanded collaboration or community features

## Teaching Note

Prospective students should read this file as a backlog of ideas, not as a promise list. A good backlog records direction while still allowing engineering reality to change priorities.
